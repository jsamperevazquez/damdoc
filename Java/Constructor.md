Este metodo se llama igual que la clase y no puede devolver nada, en caso de no ser declarado Java creará uno por defecto.

```java
public class NombreClase {
	int entero;					// Crea atributo

	public NombreClase() {
		entero = 0;				// Inicializa atributo
	}
}
```
